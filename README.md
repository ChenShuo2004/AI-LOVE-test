# WARMTH

有温度阅览室。

当前项目是一个「情侣关系复盘小游戏」Demo，支持单人测试和双人测试两个入口。

## 功能

- 单人关系复盘测试
- 双人邀请链接测试
- 关系天气
- 关系称号
- 今日复盘卡
- 给 TA 的一句话

## 开发

```bash
npm.cmd install
npm.cmd run dev
```

默认本地地址：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm.cmd run build
```

## 环境

复制 `.env.example` 为 `.env.local` 后按需修改。

当前 demo 不依赖后端服务和真实密钥。
